# Java Multi-Client Chat Application

## Description
A real-time chat application built using Java Socket Programming and Multithreading.  
Multiple clients can connect to a server and chat simultaneously.

## Features
- Multi-client support
- Real-time messaging
- Username support
- Join/Leave notifications
- Online users tracking

## Technologies Used
- Java
- Socket Programming
- Multithreading

## How to Run
1. Compile:
   javac Server.java Client.java

2. Start Server:
   java Server

3. Start Client (multiple terminals):
   java Client
